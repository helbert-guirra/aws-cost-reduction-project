# Projeto AWS - Redução de Custos

Este repositório contém um relatório conceitual sobre a utilização de serviços AWS com foco na redução de custos operacionais.

## Serviços Utilizados
- Amazon EC2
- Amazon S3
- Amazon RDS


## Objetivo
Demonstrar, de forma prática e didática, como a computação em nuvem pode reduzir custos e aumentar a eficiência das empresas.
